00. Getting started

First of all, we'd like to thank you very much for purchasing our item. If you have any questions that are beyond the scope of this help file, please feel free to email us via our user page contact form.
In order to use this template you have to upload all files from Template folder to your webhost via FTP, once you've done editing them using a text editor( you can edit them after as well ).
